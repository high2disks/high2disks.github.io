# high2disks.github.io

The homepage for the high2disks project, built with [Jekyll](https://jekyllrb.com/) and the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) theme.

## Quick Start

This site is deployed automatically via GitHub Pages. To add or edit content:

1. **Edit pages** — Markdown files in `_pages/` (e.g., `about.md`, `team.md`)
2. **Edit navigation** — `_data/navigation.yml`
3. **Edit site settings** — `_config.yml` (title, description, skin, etc.)
4. **Add blog posts** — Create files in `_posts/` named `YYYY-MM-DD-title.md`

Push changes to the `main` branch and GitHub Pages will rebuild the site automatically.

## Local Development (optional)

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000 in your browser.

## Theme Customization

Change the skin in `_config.yml` by setting `minimal_mistakes_skin` to one of:
`"default"`, `"air"`, `"aqua"`, `"contrast"`, `"dark"`, `"dirt"`, `"neon"`, `"mint"`, `"plum"`, `"sunrise"`
